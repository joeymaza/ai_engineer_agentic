```python
# Program to calculate the first 10,000 terms of the series: 1 - 1/3 + 1/5 - 1/7 + ...

# Initialize total sum
sum_series = 0.0

# Number of terms
n_terms = 10000

# Calculate the series
for n in range(n_terms):
    term = ((-1) ** n) / (2 * n + 1)  # calculate the term
    sum_series += term  # accumulate the sum

# Multiply the total by 4
final_result = sum_series * 4

# Print the result
print(final_result)
```
